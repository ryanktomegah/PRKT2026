# C1 — Failure Classifier Development

Work on the C1 Payment Failure Classifier component.

## Architecture
GraphSAGE (graph neural network) + TabTransformer (tabular deep learning) + LightGBM ensemble
- **GraphSAGE**: Learns corridor-level patterns from BIC-BIC transaction graph
- **TabTransformer**: Attention over categorical features (BIC, status, rejection code)
- **LightGBM**: Gradient boosting on engineered features (fallback + ensemble member)
- **Calibration**: Isotonic regression for probability calibration
- **SHAP**: Integrated-gradient explanations (vectorized batch)

## Performance Targets
- AUC: 0.850 (current baseline: 0.739)
- F2-score: ≥ 0.80
- Inference latency: ≤ 30ms
- Threshold τ*: 0.152 (F-beta optimized, β=2)
