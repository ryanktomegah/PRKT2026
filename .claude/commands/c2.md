# C2 — PD Model Development

Work on the C2 Probability of Default and Fee Pricing component.

## Architecture
Tiered structural PD framework + LGD estimation + fee calculation (300bps floor)
- **Tier 1**: Merton/KMV structural model (listed GSIBs with observable equity)
- **Tier 2**: Damodaran sector-median asset volatility proxy (private firms)
- **Tier 3**: Altman Z'-score → Moody's default rate table (data-sparse)

## Fee Formula (Claim 1(e))
```
Expected Loss = PD_structural × EAD × LGD × DF
CVA cost rate = EL / EAD / T (annualized)
fee_bps = max(CVA_cost + funding_spread + margin, 300)  ← floor enforced
```

## Key Differentiation from Prior Art
JPMorgan US7089207B1 covers ONLY Tier 1 (listed companies).
LIP's Tier 2 (Damodaran) and Tier 3 (Altman Z') are the core patent contribution.
These tiers handle private mid-market counterparties — the majority of cross-border trade.
